How to Set Up Discord Notifications

  1. Open Discord and go to the server where you want to receive notifications
  2. Right-click on a channel and select "Edit Channel"
  3. Click on "Integrations", then "Webhooks"
  4. Click "New Webhook", give it a name (e.g., "Heylisten")
  5. Click "Copy Webhook URL"
  6. Paste the URL in the form below
